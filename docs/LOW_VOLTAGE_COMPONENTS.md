# 弱电元件清单

## 调研来源

- Adafruit Learn Arduino 课程覆盖 LED、RGB LED、按钮、电位器、光敏电阻、蜂鸣器、LCD、直流电机、舵机、步进电机、PIR 等常见教学弱电器件。
- Arduino 文档包含电位器、晶体管电机控制、步进电机等基础电子控制说明。
- DIP switch 资料说明拨码开关常用于 PCB 上的地址、模式和配置输入。
- SparkFun Inventor's Kit 面向入门电路，覆盖 LED、电阻、电位器、按钮、光敏电阻等基础元件。

## 当前仿真抽象

当前版本仍保持稳定的直流稳态模型：导线和闭合开关合并为理想导通节点，电源正负极提供固定电位，其余可接入元件按等效负载计算端电压、电流、功率和工作强度。步进电机、舵机、继电器、传感器等已经预留 `effectKind` 和 `simulationRole`，后续可以扩展成多端子、多相线圈、PWM、方向控制、触点联动或阈值触发模型。
